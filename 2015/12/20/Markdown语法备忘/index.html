Markdown语法备忘

  • 随笔

Markdown简介

Markdwon是一种轻量级标记语言，它允许人们使用易读易写的纯文本格式编写文档，然后转换成格式丰富的HTML页面。

Markdown语法

1.标题

# H1 <一级标题>

## H2 <二级标题>

依次类推，共六级

###### H6 <六级标题>

2.文字格式

**粗体**–>在要加粗的文字左右各加两个*(星号)

*斜体*–>在要加粗的文字左右各加一个*(星号)

3.无序列表

在列表项前面加*号（*号和文字之间有空格）

* 列表项1

* 列表项2

* 列表项3

4.有序列表

在列表前面加数字，数字后面有”.”

1.列表项1

2.列表项2

3.列表项3

5.引用

>要引用的文字

6.链接

[链接名称]（链接地址）

7.添加图片

与普通链接就一个!区别

[图片地址]（图片地址）

8.换行

按两次回车或者在句末打

9.分割线

***或者—

10.添加代码

使用`<>`把代码添加到尖括号里面

11.代码高亮

```Python

def Say_hello():

print “hello world”

```

12.空格

不断行的空白写&nbsp

全方大的空白写&emsp

半方大的空白写&ensp
